Complete Setup Guide — Restaurant Analytics Dashboard

1

Install Required Libraries

Open Terminal on your Mac and run this one command:

pip install streamlit pandas numpy scikit-learn plotly matplotlib seaborn nltk openpyxl

Wait for it to finish (2-5 minutes). You'll see "Successfully installed..." when done.

2

Create Folder Structure

In Terminal, run these commands one by one:

cd /Users/komudigimhani/Desktop/SmartRetail360
mkdir -p data/raw data/processed models reports app/pages

Then copy your dataset into the right place:

cp ~/Desktop/Anonymized_Restaurant_Sales_Data.csv data/raw/

⚠️ Or drag the file manually in Finder to:
/Users/komudigimhani/Desktop/SmartRetail360/data/raw/Anonymized_Restaurant_Sales_Data.csv

3

Copy All Project Files

You received a ZIP file with all project files. Copy them like this:

SmartRetail360/ ├── pipeline.py ← Run this in Jupyter ├── requirements.txt ├── data/ │ └── raw/ │ └── Anonymized_Restaurant_Sales_Data.csv ← YOUR DATASET └── app/ ├── Home.py ← Main dashboard └── pages/ ├── 1_Sales.py ├── 2_Products.py ├── 3_Segments.py ├── 4_Forecast.py └── 5_Insights.py
4

Run the Pipeline in Jupyter Notebook

In Terminal:

cd /Users/komudigimhani/Desktop/SmartRetail360
jupyter notebook

This opens Jupyter in your browser. Then:

  • Click New → Python 3 (top right)
  • Copy the contents of pipeline.py into the notebook
  • Split into cells at each # CELL N comment using Ctrl+Shift+-
  • Run each cell with Shift+Enter
  • Wait for ✅ before running the next cell
  • Cell 2 (loading data) is fast — your CSV loads in seconds
⚠️ Run ALL 12 cells top to bottom. Do not skip any.
5

Verify All Files Were Created

After Cell 12 runs, you should see all ✅ green checkmarks. Verify in Terminal:

ls /Users/komudigimhani/Desktop/SmartRetail360/models/
ls /Users/komudigimhani/Desktop/SmartRetail360/data/

You should see:

models/ kmeans_model.pkl rfm_scaler.pkl forecast_model.pkl forecast_features.json forecast_meta.json cancel_model.pkl cancel_scaler.pkl cancel_features.json cancel_meta.json data/ smartretail.db ← The main database processed/ daily_timeseries.csv product_sentiment.csv
6

Launch the Dashboard

Open a NEW Terminal window (don't use the Jupyter one) and run:

cd /Users/komudigimhani/Desktop/SmartRetail360
streamlit run app/Home.py

Your browser will automatically open at http://localhost:8501

✅ Dashboard is working! You'll see the 🍽️ SmartRetail360 header with all metrics.
7

Navigating the Dashboard

  • 🏠 Home — KPIs, revenue trend, category charts, heatmap, top items
  • 📈 Sales Analytics — Monthly trends, day/hour analysis, delivery vs collection
  • 🍕 Product Analysis — Treemap, top items, profitability scatter, sentiment
  • 👥 RFM Segments — Order segments, 3D scatter, distributions
  • 🔮 Forecast — 30-day revenue forecast with slider
  • 💡 ML Insights — Cancellation model, business insights, profit analysis

Use the sidebar filters on any page to filter by date range, order type, and category.

❗

Troubleshooting Common Errors

Error: "Module not found"

pip install [module_name]

Error: "Database not found"

You haven't run the pipeline yet, or didn't complete all cells. Go back to Step 4.

Error: "No such file: Anonymized_Restaurant_Sales_Data.csv"

Make sure the CSV is in: data/raw/Anonymized_Restaurant_Sales_Data.csv

Dashboard won't start

cd /Users/komudigimhani/Desktop/SmartRetail360
streamlit run app/Home.py

Make sure you are IN the SmartRetail360 folder before running streamlit.

Port already in use

streamlit run app/Home.py --server.port 8502
